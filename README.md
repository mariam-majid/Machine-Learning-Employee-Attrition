# Employee Attrition Prediction

## Overview of the Project
This project predicts whether an employee is likely to leave a company using a Logistic Regression machine learning model. This project includes data cleaning, visualization, feature engineering, feature scaling, model training and model evaluation.

## Dataset
The project uses the IBM HR Employee Attrition dataset from Kaggle, which contains employee information such as age, department, monthly income, overtime, job role, and attrition status.

## Steps Performed
- Imported the required Python libraries.
- Loaded and explored the whole dataset.
- Checked for duplicate records.
- Removed unnecessary columns.
- Performed feature engineering by encoding categorical variables.
- Split the data into training and testing sets.
- Applied feature scaling using StandardScaler.
- Trained a Logistic Regression model.
- Evaluated the model using accuracy.
